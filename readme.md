# Rest API nodejs

## Description

This api is full node.js

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
```

## Test API with swagger

default port is 3000

```bash
server in http://localhost:3000/api
swagger in http://localhost:3000/swagger
```
